# headerGEN

## Description
I made headerGEN for my CS classes in which we study cpp
this is made to automate the creation of .h files

## Installation
```
git clone https://github.com/MohamedAliChabani/headerGEN.git
cd headerGEN
python3 -m venv venv
pip install pyyaml
python3 main.py
```
